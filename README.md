## Diagitizer32
Diags booting utility for 32-bit devices

## Getting started
1. Connect a device in DFU mode.
2. Press the button.
3. If successful, your device is now in Diags Mode.

NOTE: If your device is not supported, the app will say "No DFU device connected"
